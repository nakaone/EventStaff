convertCharacters(str, kana) → {string}

convertCharacters: 文字種を変換

全角英数字は半角に、半角片仮名は全角に強制的に変換。
全角ひらがな<->全角カタカナは引数(kana)で指定。既定値はひらがなに変換。

参考：
  • 全角ひらがな⇔全角カタカナの文字列変換
  • 全角⇔半角の変換を行う(英数字、カタカナ)

genChild(template, dObj, pFP) → {object}

genChild: テンプレートに差込データをセットした要素を生成
Parameters:
Name Type Description
template object 生成対象となる要素のテンプレート
Properties
Name Type Description
tag string タグ
children string 子要素。templateを再帰的に定義
text string テキスト要素。'\t'の部分はvariableの値で置換
skip string 「dObj[skip]=undefined⇒スキップ」の指定
例：参加者名が空欄なら名簿行を作成しない
variable string テキスト要素の'\t'に埋め込むdObjの要素(ex.variable='名前'⇒dObj['名前'])
max string 不定回数繰り返す場合、その最大を指定
class string クラス名。三項演算子なら評価結果()
id string setAttributeで設定
type string setAttributeで設定
name string setAttributeで設定
value string setAttributeで設定
accept string setAttributeで設定
capture string setAttributeで設定
width string setAttributeで設定
height string setAttributeで設定
style string setAttributeで設定
onclick string location.hrefする場合の遷移先URL
checked boolean trueならcheckedを追加
opt string tag="select"の場合に作成されるoptions。variableの値が選択された状態になる
dObj object 実データのオブジェクト
pFP string パン屑リストの表示名
Source:
  • library.js, line 219
Returns:
処理結果と生成されたノード
  • append {boolean} - 追加対象となる子要素ならtrue
  • status {string} - 処理結果
    • 'skipped' => テンプレートのskip条件を満たすため、生成を見送った子要素
    • 'variable' => テンプレートのtextが未指定なので、変数の値をそのままセット
    • 'replaced' => テンプレートのtext指定に基づき、プレースホルダ('\t')に変数の値をセット
    • 'fixed' => テンプレートのtext指定にプレースホルダがなく、textをそのままセット(固定文字列)
    • 'node' => セットする文字列が指定されていない要素(既定値)
  • result: 子要素のオブジェクト。エラー時はエラーオブジェクト
Type
object
Example

利用例

const template = [
  {tag:'div', class:'video', style:opt.video, children:[
    {tag:'video', style:'width:100%;'}]},
  {tag:'div', class:'camera', style:opt.camera, children:[
    {tag:'input', type:'file', accept:"image/*", capture:"camera", name:"file"}]},
  {tag:'div', class:'finder', style:opt.finder , children:[
    {tag:'canvas', style:'width:100%'},]},
]
for( let i=0 ; i<template.length ; i++ ){
  let o = genChild(template[i],{},'root');  // 全体の定義と'root'を渡す
  if( toString.call(o.result).match(/Error/) ){  // エラーObjが帰ったら
    throw o.result;
  } else if( o.append ){  // 追加フラグがtrueなら親要素に追加
    scanner.appendChild(o.result);
  }
}
